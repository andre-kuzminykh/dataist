# AIT Academy

Учебные материалы AIT в формате анимированных карточек-слайдов для соцсетей.
Каждый урок — самостоятельная папка с `index.html` (карусель квадратных слайдов 1080×1080),
спецификацией `SPEC.md` и тестами.

## Уроки
| № | Урок | Папка |
|---|------|-------|
| 1 | Loop Engineering | [`1_Loop_Engineering/`](1_Loop_Engineering/) |

Стиль: шрифт **JetBrains Mono**, фирменные цвета AIT (оранжевый `#F97316`,
фиолетовый `#8854F3`, зелёный `#10B981`, синий `#3B82F6`), обложка на тёмном фоне,
контентные слайды — на светлом, постоянная анимация на каждом слайде.
